## 프로젝트 4조
WinForm으로 턴제 카드게임을 개발하자!

Steam에 출시 되어 있는 인스크립션 [Steam link](https://store.steampowered.com/app/1092790/Inscryption/) 게임의 전투 시스템을 참고하여 만든 게임이며,  다양한 카드로 상대방 카드를  
제압하면서 Hp를 0으로 만들면 이기는 게임입니다.  

플레이 방식은 턴제 카드게임 전랙으로써 자신의 차례에 카드를 뽑아, 능력치를 갖고 있는 카드를 필드로 내 보내면서 나 자신을  
지키는 카드이자 상대를 공격하는 카드이기도 합니다.  

카드의 능력치는 체력과 공격력 2가지가 있으며, 카드의 능력치가 상대보다 높을 경우 자신의 카드가 받는 데미지는 반감 하는  
특수 효과가 존재합니다.
