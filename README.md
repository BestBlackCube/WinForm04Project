# WinForm04Project

## 프로젝트 소개
윈도우 애플리케이션을 개발하는 프레임 워크이지만, 윈도우 7 즈음 한번쯤 즐겨봤었던 핀볼 게임, 지뢰 찾기, 스파이더 카드 등  
간단한 게임을 즐겼던 기억들이 있을 겁니다.

저희 조는 이걸 중점으로 두어, 윈도우에서 간단하게 전략적으로 즐길 수 있는 게임을 개발 하면 어떨까? 라는 생각으로  
기획을 작성하여 여러가지의 카드게임의 레퍼런스 게임의 전투 시스템을 참고하여 게임을 제작하기로 정했습니다.

## 게임 소개
Steam에 출시되어 있는 인스크립션( [스팀 링크](https://store.steampowered.com/app/1092790/Inscryption/) ) 게임 전투 시스템의 방식을 참고하여 윈도우 폼으로 제작한 게임이며, 다양한 카드로  
전략을 만들어 상대를 이기는 게임입니다.

유희왕 게임의 UI를 참고하여 전투 필드와 카드 칸을 시각적으로 보여주고, 버튼 이벤트를 사용하여 반 수동적으로 게임을 플레이  
할 수 있도록 전체적인 게임 시스템을 만들고, 인스크립션 전투 시스템의 기반으로 전투 하지만, 특정 규칙을 추가하여 전투 시스템을  
개량 하였습니다.

플레이 방식은 턴제 카드게임 전랙으로써 자신의 차례에 카드를 뽑아, 능력치를 갖고 있는 카드를 필드로 내 보내면서 나 자신을  
지키는 카드이자 상대를 공격하는 카드이기도 합니다.

## 프로젝트 기획
게임 개발에 앞서 무작정 만들기 시작하면 게임 밸런스, UI, 레벨 디자인 등 차질이 생겨 붕괴가 될게 확고하니, 기획을 하여 기초틀을 잡고 나서, 밸런스를 유지 해 왔습니다.

### 팀원 역할
- ### 기획: 카드 게임 레퍼런스 분석 및 시스템 설계
    - 다양한 카드게임의 레퍼런스를 분석하여, 프로젝트에 사용할 시스템과 분석한 시스템을 응용하여 새로운 시스템을 만들어 게임로직에 복수로 사용하고, 필요한 디자인, GUI구도 등 구상하여 프로젝트를 설계한다.

- ### 게임 로직: 게임 시스템 구현, 클라이언트 로직 개발
    - 다른게임의 시스템을 벤치마킹 하여 만들어진 독창적인 게임 시스템 이론을 바탕으로 우리들만의 게임 클라잉언트 로직을 만들어 내부 이벤트을 연결하여 게임 플레이가 가능하도록 개발과 테스트를 거쳐 버그를 개선한다.

- ### 게임 밸런스: 난이도 조절 및 카드 밸런싱
    - 카드 게임에 알맞느 난이도를 구상하고, 플레이어와 적이 사용할 카드의 능력치 배분 기준을 합니다. 단일 카드로만은 게임을 이길 수 없다는 구성과 함께 여러가지의 카드를 사용하여 전략적으로 게임을 플레이 할 수 있도록 유도합니다.
- ### 그래픽 디자이너: 이미지, 배경, UI 이미지 제작
    - 기획에 알맞은 카드 이미지와 UI의 배경, 메인에 보일 배경화면등 여러가지를 제작합니다.
