# WinForm04 Project 카드 게임 개발 Final Version
## 기능 설명
발견된 버그들을 수정 후, 정식적인 개발을 중단 하고, 이후 개발된 버전들은 정식으로 개발된 버전이 아닙니다.

- ### Form1
    - 게임 플레이가 가능한 소스 코드가 들어있는 폼. 게임플레이에 사용되는 모든 기능들이 들어있다.
- ### Form2
    - 난이도를 설정하는 폼. Form1, Form4 등 연결 되어있다.
- ### Form3
    - 튜토리얼이 적혀있는 폼. Form4, Form5 등 연결 되어있다.
- ### Form4
    - 부가 기능 설정창이며, 난이도 설정, 튜토리얼, 전투 딜레이 제거, 화면 비율 설정, 폰트 변경등 기능들이  
    집약 되어있는 폼. Form1, Form5 등 연결 되어있다.
- ### Form5
    - 게임 플레이, 튜토리얼, 설정 등 이동이 가능한 메인 화면 폼. 모든 폼과 연결 되어있다.
## 버전 업데이트 지연
```
NA.NA.4v 업데이트 지연
    1.1.0v -> 1.1.1v
    - 마법 카드 사용시 캐릭터 카드에 사용하는 것이 아닌 필드에 소환되는 버그
    - 버리기 기능 사용시 카드가 없어도 없어지는 버그

    2.1.0 -> 2.1.2v 
    - 마법 카드를 사용 할때 필드 소환의 버튼이 남아 있음.
      눌렀을 시 데이터의 변화는 없음
    - 버리기 기능을 켠 상태로 손 패의 버튼이나 필드의 pictureBox를 클릭시 버리기 버튼이 활성화 되지 않음.
    - 턴 종료 시 손패 버튼이나 필드의 pirtureBox를 클릭시 일부분 버튼이 활성화 됨.
    
    2.2.0v -> 2.2.3v
```
## 버전 표기법 (Semantic Versioning)
```
[주 버전].[부 버전].[수 버전]
   0   .   0   .   0

- 주 버전: 하위 호환성이 깨지는 변경
- 부 버전: 하위 호환성 유지하며 기능 추가
- 수 버전: 하위 호환성 유지하며 버그 수정
